# ixd-5205-ixd-studio-2
Class files and exercises from IXD Studio 1 &amp; 2.

Course Code: IXD 5205

Professors: Bernie Monette and Julia Le Clair

As with IXD 5105 IXD Studio 1, this course focuses on work in multi-disciplinary, cross-program teams to solve real-world problems. Industry partners are involved, and there are opportunities for students to bring passion projects into the studio each semester with an emphasis on the development of projects that focus on "design for good" or "change for the better". 

There is an emphasis on group work and the start to finish life cycle of a project is fully explored. This course is delivered in a hands-on, experiential learning environment and employs tools and processes based on current industry standards and practices.
